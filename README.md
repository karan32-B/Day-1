Student Record System
A simple C++ program to manage student data using Object-Orientend-Programming(OOP)

Concept Covered 
. class & object 
. Member functions

Project Task 
The program create a student class withe the following details:
1) Attributes
   . Name
   . Roll Number
   . Marks
2) Functiions
   . Input Details: To take data from the user.
   . Display Details : To show the student information.
   . Calculate Grade: Automatically assigns a grade (A,B,C, or Fail).
3) Program Logic
   . Uses an Array of Objects to handle multiple students at once.

Bonus Featured Included 
   . Find Class Topper : Identifies the student with the highest marks.
   . Display Average: Calculates the average marks of the class.

How to run 
1) Copy the code into a file name stud.cpp
2) Compile the code: 
 g++ stud.cpp -o stud_project
3) Run the program :
  ./stud_project
